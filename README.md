# 🚀 Client Data Management System

This project was developed as part of my **Full-Stack Web Development** studies. The system focuses on management, persistence, and security of user data and access logs.

## 🛠️ Technologies Used

* **Node.js**: JavaScript runtime environment.
* **Express**: Framework for building APIs and handling routes.
* **Bcrypt**: Library for password hashing and security.
* **JavaScript (ES6+)**: Core application logic.
* **File System (FS)**: For local data storage and file manipulation.

## 📂 Project Structure

* `server.js`: Entry point of the application.
* `models/User.js`: User data structuring logic.
* `acessos/`: Directory for storing access logs and records.
* `.gitignore`: Configuration to prevent tracking of heavy files (`node_modules`) and sensitive data.
